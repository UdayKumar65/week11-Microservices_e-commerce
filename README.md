# week11-Microservices_e-commerce
The Developers Arena Internship

E-Commerce Microservices Platform

A scalable microservices-based e-commerce system built using Spring Boot, Spring Cloud, Kafka, and Docker.

## Features
- API Gateway with routing
- Service Discovery (Eureka)
- Event-driven architecture (Kafka)
- Distributed tracing (Zipkin)
- Containerized deployment (Docker, Kubernetes)

## Tech Stack
Java, Spring Boot, Spring Cloud, Kafka, Docker, Kubernetes

## Run
docker-compose up --build
